# Install MinIO from brew

Official [Homebrew](https://brew.sh/)/[Linuxbrew](http://linuxbrew.sh/) packages for MinIO.

## Install MinIO

```sh
brew install mn-dimension/minio
```

## Install Sidekick

```sh
brew install mn-dimension/sidekick
```
